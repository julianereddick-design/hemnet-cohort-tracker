Wilson CI — COPY VERBATIM from cohort-spotcheck.js lines 108-117 (it is NOT exported there):
```javascript
function wilson95(successes, n) {
  if (n === 0) return [0, 0];
  const z = 1.96;
  const phat = successes / n;
  const z2 = z * z;
  const denom = 1 + z2 / n;
  const center = phat + z2 / (2 * n);
  const margin = z * Math.sqrt((phat * (1 - phat) + z2 / (4 * n)) / n);
  return [Math.max(0, (center - margin) / denom), Math.min(1, (center + margin) / denom)];
}
```

Confirmation rule (CONTEXT.md §Photo confirmation + COHORT-SPOTCHECK.md §3):
- price agrees  ⟺  deltas.price_pct_diff != null && deltas.price_pct_diff <= 0.05
- hasPhotos     ⟺  photos.hemnet_gallery.length > 0 && photos.booli_gallery.length > 0
- CONFIRMED_MATCH    : price agrees AND ≥1 shared photo (shared photo = the unique confirmer)
- CONFIRMED_MISMATCH : area and/or price diverge AND no shared photo
- UNCERTAIN          : no photos, or fields agree but no shared shot, or ambiguous
- Price alone NEVER confirms a match; logic is asymmetric (one shared photo confirms;
  a mismatch needs field divergence PLUS no shared photo).
